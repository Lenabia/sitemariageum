# Dossier Images

Placez vos images ici.

## Image de fond principale

Pour l'image de fond du site :
1. Copiez votre image dans ce dossier
2. Nommez-la `couple-photo.jpg` (ou modifiez le nom dans `WeddingSite.jsx`)
3. Formats supportés : `.jpg`, `.jpeg`, `.png`, `.webp`

## Nom du fichier

Par défaut, le code cherche : `couple-photo.jpg`

Si votre image a un autre nom, modifiez la ligne dans `WeddingSite.jsx` :
```jsx
import couplePhoto from '../assets/images/VOTRE_NOM_IMAGE.jpg';
```
